# sudoku-solver
An application solving a sudoku puzzle using a graph database as a data model.

## Linting

```
./gradlew lint
```

## Formatting

```
./gradlew format
```

## Testing

```
./gradlew test -i
```

## Building

```
./gradlew build
```

## Running

```
./gradlew run --args="<file-with-initial-grid>"
```
